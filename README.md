## Install
``` bash
git clone https://github.com/SapienNetwork/react-component-boilerplate
yarn install
```

## Storybook
```bash
# serve the components at localhost:5000
yarn storybook
```

## Develop
```bash
# serve the app at localhost:8080
yarn dev
```

## Build
```bash
# build for production
yarn build
```

## Lint
```bash
# lint all the things
yarn lint
```
